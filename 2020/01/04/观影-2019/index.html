影评, 总结, 观影

一、2019才看过的

按照推荐力度从先到后排序。

一一 (2000)
2000 / 中国台湾 日本 / 剧情 爱情 家庭 / 杨德昌 / 吴念真 李凯莉

电影艺术一例。这部电影的话题比较沉重：人为什么而活。我还不算有重重牵绊的中年人，但也到了能看懂这部电影的年纪。

我们与恶的距离 (2019)
2019 / 中国台湾 / 剧情 / 林君阳 / 贾静雯 温升豪

令我震撼的台剧，讨论社会、家庭的问题，相当深刻。这个世界需要理解，需要关怀，需要爱。“愿你的生命中始终有光，愿你成为一束光。”

寄生虫 (2019)
2019 / 韩国 / 剧情 喜剧 / 奉俊昊 / 宋康昊 李善均

电影艺术一例。一个精彩、充满悬念的故事，情节可真谓跌宕起伏。电影的节奏控制了我的呼吸和心跳。电影对于贫富阶级的冲突也有较多讨论。

大鱼 (2003)
2003 / 美国 / 剧情 家庭 奇幻 / 蒂姆·波顿 / 伊万·麦克格雷格 阿尔伯特·芬尼

一个奇幻、浪漫的故事。先设想自己想以什么方式死去，再去思考如何度过这一生。“一个人只拥有此生此世是不够的，他还应该拥有诗意的世界。”

绿皮书 (2018)
2018 / 美国 中国大陆 / 剧情 喜剧 传记 / 彼得·法雷里 / 维果·莫腾森 马赫沙拉·阿里

一个充满温情的故事。在了解了美国种族隔离的历史之后，看起来更有感触。

非自然死亡 (2018)
2018 / 日本 / 剧情 悬疑 / 冢原亚由子 竹村谦太郎 村尾嘉昭 / 石原里美 井浦新

很好看的日剧，温暖、治愈。

死亡诗社 (1989)
1989 / 美国 / 剧情 / 彼得·威尔 / 罗宾·威廉姆斯 罗伯特·肖恩·莱纳德

我们读诗、写诗并不是因为它们好玩，而是因为我们是人类的一分子，而人类是充满激情的。人生短暂，要珍惜。这个世界有太多的利益纠结，学校可以为了自身利益而枉顾学生的人格发展，磨灭人性之举可以毁人一生，这样的事情发生在每一个角落。一个人要独善其身都尚且困难，更何况还要擎着火把指引后辈之路，这样的老师可谓伟大。

辩护人 (2013)
2013 / 韩国 / 剧情 / 杨宇硕 / 宋康昊 金英爱

小人物被推到时代的潮头，人心正义的力量在挣扎。韩国人真敢拍，宋康昊 rocks。

波西米亚狂想曲 (2018)
2018 / 英国 美国 / 剧情 传记 同性 / 布莱恩·辛格 / 拉米·马雷克 本·哈迪

Queen、Freddy的传记性电影，我因为这部电影而开始爱上Queen。

杀人回忆 (2003)
2003 / 韩国 / 剧情 动作 犯罪 / 奉俊昊 / 宋康昊 金相庆

年初看的，是那种看完后会偶尔想起的电影。宋康昊 rocks。

头号玩家 (2018)
2018 / 美国 / 动作 科幻 冒险 / 史蒂文·斯皮尔伯格 / 泰伊·谢里丹 奥利维亚·库克

好玩。

二、 以前就看过的电影

肖申克的救赎 (1994)
1994 / 美国 / 犯罪 剧情 / 弗兰克·德拉邦特 / 蒂姆·罗宾斯 摩根·弗里曼

每年都要看一遍系列。无论身处何地何境，心中都要怀有希望。2019的一个新感触，音乐或者说艺术作品对于人的意义。那些会在脑海中出现的旋律和画面，可以变成沙漠中的清泉。

霸王别姬 (1993)
1993 / 中国大陆 中国香港 / 剧情 爱情 同性 / 陈凯歌 / 张国荣 张丰毅

每年都要看一遍系列。个人在时代面前如此渺小，仿佛洪流中被裹挟着的一粒砂石。人与人之间的真情，在社会大潮中脆弱不堪，因而更加弥足珍贵。

阿甘正传 (1994)
1994 / 美国 / 剧情 爱情 / 罗伯特·泽米吉斯 / 汤姆·汉克斯 罗宾·怀特

温暖、治愈、轻松、快乐，结合美国的现代历史看更有意思。

心灵捕手 (1997)
1997 / 美国 / 剧情 / 格斯·范·桑特 / 马特·达蒙 罗宾·威廉姆斯

讨论成长与爱的电影，让我对宽容、理解和爱有更深的认识。

当幸福来敲门 (2006)
2006 / 美国 / 剧情 传记 家庭 / 加布里埃莱·穆奇诺 / 威尔·史密斯 贾登·史密斯

勇敢地追求梦想吧，所有为了美好梦想而付出的努力都不会白费。

低俗小说 (1994)
1994 / 美国 / 剧情 喜剧 犯罪 / 昆汀·塔伦蒂诺 / 约翰·特拉沃尔塔 乌玛·瑟曼

电影艺术一例。电影的讲故事手法很特别，值得一看再看。

十二怒汉 (1957)
1957 / 美国 / 剧情 / 西德尼·吕美特 / 亨利·方达 马丁·鲍尔萨姆

电影艺术一例。把小小的会议室里，十二个男人的唇舌之战，拍出了巨大的张力。不愧是经典。

黑客帝国 (1999)
1999 / 美国 澳大利亚 / 动作 科幻 / 莉莉·沃卓斯基 拉娜·沃卓斯基 / 基努·里维斯 劳伦斯·菲什伯恩

内核坚硬，情节精彩。“如何证明自己是在现实世界而非做梦”。

星际穿越 (2014)
2014 / 美国 英国 加拿大 / 剧情 科幻 冒险 / 克里斯托弗·诺兰 / 马修·麦康纳 安妮·海瑟薇

科学伟大，亲情感人。马修·麦康纳 rocks。

沉默的羔羊 (1991)
1991 / 美国 / 剧情 犯罪 惊悚 / 乔纳森·戴米 / 朱迪·福斯特 安东尼·霍普金斯

朱迪·福斯特的表演令人难忘。

三、还不错的电影

头号玩家 (2018)
2018 / 美国 / 动作 科幻 冒险 / 史蒂文·斯皮尔伯格 / 泰伊·谢里丹 奥利维亚·库克

商业片典范之作。

复仇者联盟4：终局之战 (2019)
2019 / 美国 / 剧情 动作 科幻 / 安东尼·罗素 乔·罗素 / 小罗伯特·唐尼 克里斯·埃文斯

商业片典范之作。

流浪地球 (2019)
2019 / 中国大陆 / 科幻 冒险 灾难 / 郭帆 / 屈楚萧 吴京

不输于好莱坞的特效制作，希望有更多刘慈欣的作品可以上荧幕。情节不错，但是价值观不合我胃口。

四、我觉得一般的电影

千与千寻 (2001)
2001 / 日本 / 剧情 动画 奇幻 / 宫崎骏 / 柊瑠美 入野自由

今年去看了院线版，感觉低于预期，可能真是因为自己过了那个年纪了。

搏击俱乐部 (1999)
1999 / 美国 德国 意大利 / 剧情 动作 悬疑 / 大卫·芬奇 / 爱德华·诺顿 布拉德·皮特

过了能对这部电影有深度共鸣的年纪了，电影色调比较阴暗，需要调高亮度观看。

蝙蝠侠：黑暗骑士 (2008)
2008 / 美国 英国 / 剧情 动作 科幻 / 克里斯托弗·诺兰 / 克里斯蒂安·贝尔 希斯·莱杰

6年前看觉得还蛮不错，今年再看没有太多感触。
